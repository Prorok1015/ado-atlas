## Одним абзацем

Инженерное качество ядра — заметно выше среднего для соло-проекта: чистое разделение
чистых функций и IO, продуманный ретрай с `Retry-After`, аккуратный WIQL-квотинг,
«секретный файрвол» в `App.prefs`, честная `ARCHITECTURE.md` с зафиксированными
трейд-оффами. Проблема не в качестве кода, а в том, что **продуктовый контур не замкнут**:
премиум-витрина с ценником рекламирует фичи, которых нет, метод `gate()` не вызывается
ни разу, а `activate()` бросает исключение. Продать Pro сегодня физически нельзя, а если
бы было можно — купивший всё равно упёрся бы в пейволл. Второй пласт — тихие потери
данных (записи без проверки ревизии, ретрай неидемпотентных POST) и нулевая доступность
для незрячих. Всё это чинится инкрементально; см. [бэклог](11-backlog.md).
