# Сводный бэклог

Аудит и роадмап в одном приоритизированном списке. Это то, с чем можно работать
изо дня в день, не перечитывая остальные документы.

Обозначения: `[A]` — из аудита, `[R]` — из роадмапа.
Оценки — для соло-разработчика, знакомого с кодовой базой.

## Если времени совсем нет

Три вещи, без которых не имеет смысла остальное:

1. **`gate()`** (1.1) — иначе платящий пользователь не получит того, за что заплатил.
2. **CI + починить `ai.test.js`** (0.1, 0.2) — иначе ты не узнаешь, что что-то сломал.
3. **Убрать ценник с несуществующего** (0.3) — иначе можно потерять канал продаж целиком.

Полтора дня работы суммарно.
