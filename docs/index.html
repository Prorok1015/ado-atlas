ADO Atlas

See and manage your Azure DevOps work items as a tree, dependency graph, kanban board, and sprint Gantt — right in your browser.

Privacy Policy Microsoft sign-in setup Source on GitHub

What it does

  • Tree / Graph / Board / Sprint Gantt views over your work items.
  • Inline editor: state, assignee, priority, sprint, dates, description, re-parenting, comments & history.
  • Filters (State, Type, Priority, Assignee, Sprint, Tags), bulk edit, command palette, CSV/JSON export.
  • Talks directly to the Azure DevOps REST API with your Personal Access Token — no backend.

Privacy at a glance

Your token and data stay on your device (Chrome storage.local). The extension only ever contacts Azure DevOps (dev.azure.com and app.vssps.visualstudio.com) — there is no analytics, no tracking, and no third-party server. Full details in the Privacy Policy.

ADO Atlas's use of data adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.
