# План работ: Поддержка упоминаний пользователей через «@»

## 1. Описание задачи
Упоминания пользователей через символ `@` в Azure DevOps отправляют уведомления на почту и подсвечивают их в интерфейсе. Для этого в ADO используется специальный формат ссылки: `<a href="#" data-vss-mention="version:2.0,id:DESCRIPTOR">@Имя</a>`. В текущей версии расширения выбор пользователей из Identity Picker при вводе `@` работает только в текстовом поле описания (`s_desc`). Необходимо расширить этот функционал на поле критериев приемки (`s_ac`) и текстовое поле добавления комментариев (`cm_text`), а также убедиться в правильности CSS-стилизации отображения ссылок-упоминаний.

1. **Автозаполнение в комментариях**:

   - Убедиться, что текст заменился на `@[Имя](дескриптор)`.
